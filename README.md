# SmallStreet Markets 
### South Africa's C2C E-Commerce Platform

A full-stack customer-to-customer (C2C) e-commerce web application built for the South African market, enabling individuals to buy and sell products online. Built as a first independent project using PHP, MySQL, and vanilla HTML/CSS/JS.

🔗 **Live Demo:** [c2c-project.lovestoblog.com/c2c-ecommerce](https://c2c-project.lovestoblog.com/c2c-ecommerce)

---

## Features

- **Product Listings** — sellers can list products with images, descriptions, and pricing in ZAR
- **Category Browsing** — products organised by category for easy discovery
- **Search** — real-time search across product titles and descriptions
- **User Authentication** — secure registration and login with session management
- **Shopping Cart** — add, update, and remove items with persistent cart state
- **Checkout** — end-to-end order flow from cart to order confirmation
- **Admin** — Oversees and manages orders of clients and approving listings of sellers

## Database

The project uses a relational MySQL database with tables for:
- `users` — registered accounts
- `products` — listings with category, price, and image path
- `categories` — product classification
- `cart` — session-linked cart items
- `orders` — completed transactions
- and more.....

## Project Context

SmallStreet Markets was built as a first independent full-stack project during my BSc Computer Science studies at Eduvos. It demonstrates end-to-end web development — from database design and server-side PHP logic to frontend UI and user session management — without relying on any frameworks.
